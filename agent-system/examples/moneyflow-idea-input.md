# Idea Input

Build an app that records transactions when a user enters:

- amount used
- merchant or usage place
- payment method such as credit card, checking account, or debit card
- source of funds

The app should:

- store the transaction history
- show daily money flow with `+` and `-`
- group the flow by the money source
- work well on mobile

The first version can start with manual input before bank sync automation.
